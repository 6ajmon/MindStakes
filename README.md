# MindStakes
MindStakes is a tool for quiz show hosts made in Godot. 

# Features:
- tracking participants score
- getting random categories
- getting random nonrepeating questions
- tracking answer time
- questions with audio/sound
- three game modes
- questions pools to choose from

### It is still a work in progress.

What I want to add:
- custom category creator
- custom questions creator
- questions import and export
- multi (dual) language support
- winner screen summary
